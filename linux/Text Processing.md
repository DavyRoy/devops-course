5. Text Processing — Обработка текста

Инструменты для чтения, фильтрации, преобразования текстовых данных в Linux.

⸻

5.1 stdout / stdin / stderr — Стандартные потоки

Подтемы:
•	stdin (0) — стандартный ввод
•	stdout (1) — стандартный вывод
•	stderr (2) — вывод ошибок
•	Перенаправления:
•	> — вывести в файл
•	2> — ошибки в файл
•	&> — всё в файл
•	< file — подать файл как ввод

⸻

5.2 cut — Вырезание колонок

Подтемы:
•	-d — указать разделитель
•	-f — выбрать поля
•	Пример: cut -d':' -f1 /etc/passwd
•	Работа с CSV/логами

⸻

5.3 paste — Склеивание строк

Подтемы:
•	Склеивание строк из разных файлов
•	По умолчанию разделитель — таб
•	Изменение разделителя: -d
•	Пример: paste file1 file2

⸻

5.4 sort — Сортировка

Подтемы:
•	Лексикографическая сортировка
•	Числовая сортировка: -n
•	Обратная: -r
•	Удаление повторов: -u
•	По колонке: -k

⸻

5.5 tr — Замена символов

Подтемы:
•	Замена символов: tr a-z A-Z
•	Удаление символов: -d
•	Сжатие повторов: -s

⸻

5.6 head — Первые строки

Подтемы:
•	По умолчанию 10 строк
•	-n для указания количества
•	Быстрый просмотр начала файла

⸻

5.7 tail — Последние строки

Подтемы:
•	По умолчанию 10 строк
•	tail -n 20 file
•	Просмотр логов в реальном времени:
•	tail -f logfile

⸻

5.8 join — Соединение файлов по колонке

Подтемы:
•	Аналог SQL JOIN для текстов
•	Объединение по ключевой колонке
•	-1 и -2 — номер ключевой колонки
•	Требует отсортированных файлов

⸻

5.9 split — Разделение файла

Подтемы:
•	Разделение по размеру или строкам
•	split -l 1000 file — каждые 1000 строк
•	split -b 1M file — по 1 МБ

⸻

5.10 pipe — Конвейеры (|)

Подтемы:
•	Передача результата одной команды другой
•	Комбинирование утилит:
•	grep error file | wc -l
•	Построение цепочек

⸻

5.11 tee — Дублирование вывода

Подтемы:
•	Показывает вывод на экран и в файл
•	command | tee file
•	Добавить: tee -a file

⸻

5.12 nl — Нумерация строк

Подтемы:
•	Добавление номеров строк
•	В отличие от cat -n, даёт гибкие настройки

⸻

5.13 wc — Подсчёт строк/слов/символов

Подтемы:
•	wc -l — строки
•	wc -w — слова
•	wc -c — байты

⸻

5.14 expand — Таб → пробелы

Подтемы:
•	Преобразует табы в N пробелов
•	expand -t 4 file

⸻

5.15 unexpand — Пробелы → табы

Подтемы:
•	Обратная операция expand
•	unexpand -t 4 file

⸻

5.16 uniq — Удаление повторяющихся строк

Подтемы:
•	Требует предварительной сортировки (sort)
•	uniq — убрать подряд идущие дубли
•	uniq -c — посчитать дубль-строки

⸻

5.17 grep — Поиск по тексту

Подтемы:
•	Простые шаблоны
•	Регулярные выражения: grep -E
•	Поиск в каталоге: grep -r
•	Подсветка совпадений: --color
•	Поиск слова целиком: -w

⸻

5.18 awk — Обработка текстовых данных (мини-язык)

Подтемы:
•	Работа с колонками: $1, $2
•	Задание разделителя: -F
•	Фильтрация строк
•	Простейшие программы:
•	awk '{print $1}' file
•	Условные проверки
•	Суммирование колонок
