2.1 Vim — Редактор Vim

Подтемы:
•	Режимы Vim:
•	Normal mode — основной режим
•	Insert mode — ввод текста
•	Command mode — команды :w, :q и т.д.
•	Основные команды:
•	i — перейти в режим вставки
•	Esc — выход в normal
•	:w — сохранить
•	:q — выйти
•	:wq — сохранить и выйти
•	dd — удалить строку
•	yy — копировать строку
•	p — вставить
•	Навигация по файлу:
•	h/j/k/l — перемещение
•	gg — начало файла
•	G — конец файла
•	Поиск:
•	/текст — поиск
•	n/N — следующее/предыдущее совпадение

⸻

2.2 Nano — Редактор Nano

Подтемы:
•	Запуск редактора:
•	nano file.txt
•	Основные функции:
•	Печать текста
•	Быстрое редактирование в одном режиме (нет режимов как в Vim)
•	Комбинации клавиш:
•	Ctrl + O — сохранить
•	Ctrl + X — выйти
•	Ctrl + W — поиск
•	Ctrl + K — вырезать строку
•	Ctrl + U — вставить строку
•	Навигация:
•	Стрелки вверх/вниз
•	Ctrl + _ — перейти к строке
